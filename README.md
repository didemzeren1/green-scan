# green-scan
Greenscan, ürünlerin barkodlarıyla karbon ayak izini ve eko-skorunu takip etmeyi sağlayan bir Spring Boot REST API projesidir. MongoDB veritabanı kullanır. Ürün ekleme ve barkoda göre çevresel etki verilerini sorgulama imkanı sunarak sürdürülebilir tüketime yardımcı olan bir arka uç servisidir.
